# HVAC Flow Rate Calculator — Quick-Start Checklist

Use this checklist to go from downloaded file to deployed tool in under 10 minutes.

---

## Before You Start

- [ ] Download `HVAC_flow_rate_calculator.html` to your computer
- [ ] Open the file in a plain-text editor (Notepad, VS Code, TextEdit, Notepad++, etc.)
- [ ] Have your company name, initials/logo, city, and certifications ready

---

## Step 1 — Customize Your Branding (4 edits)

- [ ] **Line 6** — Change `<title>HVAC Flow Rate Calculator — Your HVAC Co.</title>` to your company name
- [ ] **Line 516** — Change `HC` (the logo mark) to your initials or abbreviation
- [ ] **Line 524** — Change `Your HVAC Co.` (footer company name) to your company name
- [ ] **Line 524** — Change `Rogers, AR` to your city and state
- [ ] **Line 525** — Change `EPA 608 Universal · NATE Certified` to your actual credentials

> **Tip:** Use Find & Replace (Ctrl+H / Cmd+H) to catch every instance of "Your HVAC Co." at once.

---

## Step 2 — Test It Locally

- [ ] Save the file after your edits
- [ ] Double-click the `.html` file to open it in your browser
- [ ] Confirm your company name appears in the header and footer
- [ ] Run a test calculation in the CFM tab — verify results update live
- [ ] Add a zone in the Duct Sizing tab — confirm the table expands
- [ ] Resize your browser window narrow (or test on a phone) — confirm the layout holds

---

## Step 3 — Deploy

Choose one of the following:

- [ ] **Website:** Upload the `.html` file to your web server or hosting provider. Link to it from your website's "Tools" or "Resources" page.
- [ ] **Shared Drive:** Drop the file in a Google Drive or Dropbox folder shared with your team. Anyone with the link can open it in their browser.
- [ ] **Email to Clients:** Attach the file directly to an email. Recipients open it in any browser — no installation required.
- [ ] **Tablet/Field Use:** Save the file to a tablet's local storage or Files app. Open in Safari, Chrome, or Firefox. Works offline after the Google Fonts load once.

---

## Step 4 — Optional Polish

- [ ] Rename the file to something client-friendly (e.g., `AcmeHVAC-FlowCalc.html`)
- [ ] Add a bookmark on your field tablet for one-tap access
- [ ] If hosted on a website, test the URL on your phone before sharing it with customers

---

## You're Done

Your branded HVAC Flow Rate Calculator is live. No server setup. No subscription. No ongoing maintenance required.
